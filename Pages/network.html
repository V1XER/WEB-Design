NETWORK

  • Server: A Server is a computer that provides information to other computers called "clients" on computer network. This architecture is called the client server model.
  • Webserver: A Web server is computer software and underlying hardware that accepts requests via HTTP or its secure variant HTTPS.
  • DNS Server: DNS Server translate domain names into IP addresses, enabling DNS clients to reach the origin server.
  • IP: An Internet Protocol (IP) address is the unique identifying number assigned to every device connected to the internet.
  • Domain: A Domain Name is a unique, easy to remember address used to access websites, such as 'google.com', and 'facebook.com'.
